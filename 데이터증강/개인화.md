# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡의 인지왜곡 유형 중
[개인화 (Personalization)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
개인화란,
여러 요인이나 타인의 선택, 우연, 환경 등이 작용한 결과를
자신의 책임이나 잘못으로 과도하게 귀속시키는 사고이다.

# 표현 가이드:
- 결과나 문제가 ‘나 혼자’의 책임처럼 표현되어야 함
- "내 탓이야", "내가 문제였어", "다 나 때문에"
  와 같은 책임 귀속 표현이 자연스럽게 드러날 것
- 실제로 통제 불가능하거나 공동 책임일 수 있는 상황일 것
- 자기 자신을 고정된 단어로 규정하지 말 것 (→ 낙인찍기 금지)
- 규칙·의무 기준을 강조하지 말 것 (→ 당위적 사고 금지)
- 반복 경험을 근거로 일반화하지 말 것 (→ 과잉일반화 금지)

# 인지왜곡 예시:
- “팀이 실패한 건 다 내 탓이야.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
