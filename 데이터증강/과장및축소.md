# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

목표:
아론 벡의 인지왜곡 유형 중
[과장 및 축소 (Magnification / Minimization)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
과장 및 축소란,
부정적인 요소나 실수는 실제보다 훨씬 크게 받아들이고
긍정적인 요소나 강점은 중요하지 않은 것처럼 축소하는 사고이다.

# 표현 가이드:
- 한 문장 안에서 다음 중 하나의 구조를 가질 것
  ① 과장: 사소한 문제나 실수를 큰 문제처럼 표현
  ② 축소: 분명한 성과나 장점을 가치 없거나 하찮게 표현
- "엄청", "완전", "큰일", "아무것도 아니야"
  와 같은 크기·중요도 왜곡 표현이 자연스럽게 드러날 것
- 단순한 낙담이나 부정적 감정 표현에 그치지 말고
  ‘중요도 평가의 왜곡’이 명확히 드러나야 함
- 긍정 결과를 운이나 우연으로 돌리지 말 것 (→ 긍정 무시 금지)
- 극단적 흑백 판단이나 일반화로 넘어가지 말 것

# 인지왜곡 예시:
- “그건 아무 가치도 없어.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
