# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡의 인지왜곡 유형 중
[당위적 사고 (Should Statements)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
당위적 사고란,
자신이나 타인, 세상에 대해
“반드시 ~해야 한다”, “~면 안 된다”와 같은
경직된 규칙과 의무 기준을 적용하는 사고이다.

# 표현 가이드:
- 문장에 반드시 당위적 표현이 직접적으로 등장해야 함
  (예: 해야 해, 하면 안 돼, 당연히, 무조건)
- 규칙이나 기준 자체가 핵심이 되도록 할 것
- 기준을 지키지 못했을 때의 평가나 처벌 표현은 포함하지 말 것
  (→ 낙인찍기·과장/축소 금지)
- 반복 경험을 근거로 일반화하지 말 것 (→ 과잉일반화 금지)
- 감정이 사실의 근거로 작동하지 않도록 할 것 (→ 감정적 추론 금지)

# 인지왜곡 예시:
- “항상 완벽해야 해.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
