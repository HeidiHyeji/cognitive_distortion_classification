# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡(Aaron Beck)의 인지왜곡 유형 중
[과잉일반화]하는 문장을 머신러닝 라벨링 데이터로 사용하기 위해 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
과잉일반화는
한두 번의 실패나 사건을 근거로
앞으로도 항상 그럴 것이라고 일반화하는 사고이다.

# 표현 가이드:
- "이번에도", "또", "항상 그랬어", "어차피"
  와 같은 반복·일반화 표현이 자주 나타나도록 할 것
- 단일 사건을 전체 인생이나 미래로 확대하는 구조를 가질 것
- 낙인찍기(나는 ○○야)로 끝나지 않도록 주의

# 인지왜곡 예시:
- “한 번 실수했으니 앞으로도 다 틀릴 거야.”

# 출력 형식:
- CSV 파일
- 각 항목은 다음 필드를 가짐
  {
    "text": "...",
    "label": "과잉일반화"
	}

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
