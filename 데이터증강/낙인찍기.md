# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡의 인지왜곡 유형 중
[낙인찍기 (Labeling)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
낙인찍기란,
구체적인 행동이나 상황을 넘어
자신이나 타인을 하나의 고정된 단어 또는 표현으로
정체성처럼 규정하는 사고이다.

# 표현 가이드:
- 문장의 핵심이 반드시
  "나는 ○○야", "난 완전 ○○야",
  "그 사람은 ○○야" 와 같은
  정체성 규정 표현이어야 함
- 사건 설명은 최소화하거나 생략할 것
- 단일 실패를 근거로 일반화하지 말 것 (→ 과잉일반화 금지)
- 규칙·의무 기준을 강조하지 말 것 (→ 당위적 사고 금지)
- 감정만을 근거로 결론을 내리지 말 것 (→ 감정적 추론 금지)

# 인지왜곡 예시:
- “나는 루저야.”
- “그 사람은 최악이야.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
