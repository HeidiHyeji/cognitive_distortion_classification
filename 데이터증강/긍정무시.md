# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

목표:
아론 벡의 인지왜곡 유형 중
[긍정 무시 (Disqualifying the Positive)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
긍정 무시란,
실제로 긍정적인 결과나 성취, 칭찬이 있었음에도 불구하고
그 가치를 인정하지 않고
운, 우연, 상황 덕분이라고 평가절하하는 사고이다.

# 표현 가이드:
- 문장에는 반드시 긍정적인 결과, 성취, 또는 타인의 칭찬이 등장해야 함
- 화자는 그 긍정적 요소를 인정하지 않고
  "그냥 운이었어", "우연이었을 뿐이야", "별거 아니야"
  와 같이 의미를 깎아내리는 흐름을 가질 것
- 긍정 요소를 보지 않거나 언급하지 않는 구조는 금지 (→ 정신적 여과 금지)
- 자기 자신을 하나의 단어로 규정하지 말 것 (→ 낙인찍기 금지)
- 극단적 일반화나 흑백 판단을 포함하지 말 것


# 인지왜곡 예시:
- “그건 그냥 운이 좋았던 거야.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
