# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡(Aaron Beck)의 인지왜곡 유형 중
[정신적 여과]하는 문장을 머신러닝 라벨링 데이터로 사용하기 위해 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
정신적 여과란,
전체 상황 속에서 긍정적이거나 중립적인 요소들은 무시하고
오직 한두 가지 부정적인 요소만 선택적으로 집중하여
전체를 부정적으로 판단하는 사고이다.

# 표현 가이드:
- 긍정적인 상황이나 결과가 배경에 존재해야 함
- 그러나 화자는 그것을 거의 언급하지 않거나 무시하고
  특정 부정적 요소 하나에만 집착하는 흐름을 가질 것
- "그 한 가지 때문에", "그게 계속 걸려", "그 생각밖에 안 나"
  와 같은 선택적 집중 표현이 자연스럽게 드러나도록 할 것
- 긍정적인 요소를 운이나 우연으로 깎아내리지 말 것 (→ 긍정 무시 금지)
- 자기 자신을 하나의 단어로 규정하지 말 것 (→ 낙인찍기 금지)

# 인지왜곡 예시:
- “칭찬은 다 거짓말이야.”

# 출력 형식:
- CSV 파일
- 각 항목은 다음 필드를 가짐
  {
    "text": "...",
    "label": "정신적 여과"
	}

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
