# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

목표:
아론 벡의 인지왜곡 유형 중
[성급한 결론 (Jumping to Conclusions)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
성급한 결론이란,
충분한 증거나 사실 확인 없이
자신이 추측한 생각이나 미래 결과를
이미 확정된 사실처럼 받아들이는 사고이다.
이에는 생각 읽기와 미래 예측이 포함된다.

# 표현 가이드:
- 객관적 근거, 확인된 사실, 직접적인 말은 등장하지 말 것
- "분명히", "틀림없이", "~일 게 뻔해", "~할 거야"
  와 같은 확신 표현이 자연스럽게 포함될 것
- 다음 두 유형 중 하나의 구조를 가질 것
  ① 생각 읽기: 타인이 속으로 어떻게 생각할지 단정
  ② 미래 예측: 아직 일어나지 않은 결과를 부정적으로 단정
- 자기 자신을 하나의 단어로 규정하지 말 것 (→ 낙인찍기 금지)
- 반복 사건을 근거로 일반화하지 말 것 (→ 과잉일반화 금지)

# 인지왜곡 예시:
- “그 사람은 날 싫어할 거야.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
