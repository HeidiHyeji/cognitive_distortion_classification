# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡(Aaron Beck)의 인지왜곡 유형 중
[이분법적 사고]에 해당하는 문장을 머신러닝 라벨링 데이터로 사용하기 위해 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 표현 가이드:
- "항상", "절대", "완벽하지 않으면", "전부 아니면 아무것도"
  와 같은 극단적 표현이 자연스럽게 드러나도록 할 것
- 중간 단계, 회색지대, 예외를 인정하는 표현은 포함하지 말 것

# 인지왜곡 정의:
[이분법적 사고]
모든 일을 흑백으로만 판단하며,
중간 단계나 회색지대를 인정하지 않는 사고.

# 출력 형식:
- JSON 배열
- 각 항목은 다음 필드를 가짐
  {
    "text": "...",
    "label": "이분법적 사고"
  }
- JSON 파일로 답변 줄 것

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
