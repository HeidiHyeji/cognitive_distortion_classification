# 역할
너는 임상심리학 기반 인지행동치료(CBT) 전문가이자,
자연어처리(NLP) 학습용 데이터셋을 설계하는 데이터 사이언티스트다.

# 목표:
아론 벡의 인지왜곡 유형 중
[감정적 추론 (Emotional Reasoning)]에 해당하는 문장을
머신러닝 학습용 라벨링 데이터로 생성하라.

# 출력 조건:
- 문장은 반드시 1문장
- 한국어 구어체 또는 독백 형태
- 실제 사람이 혼잣말로 할 법한 자연스러운 표현
- 감정이 드러나되 과도한 설명이나 해설은 포함하지 말 것
- 다른 인지왜곡 유형과 혼합되지 않도록 주의할 것
- 동일한 의미/표현 반복 금지
- 상담, 치료, 인지왜곡이라는 단어 사용 금지
- 긍정적 반전이나 교정 문장 포함 금지

# 인지왜곡 정의:
감정적 추론이란,
자신이 느끼는 감정(불안, 두려움, 찝찝함 등)을
객관적 사실이나 증거처럼 받아들이고
그 감정을 근거로 결론을 내리는 사고이다.

# 표현 가이드:
- 문장에 반드시 감정 표현이 등장해야 함
  (예: 불안하다, 찝찝하다, 두렵다, 마음이 불편하다)
- 감정 → 결론 구조가 명확히 드러나야 함
  (예: “~해서 불안하다” ❌ / “불안하니까 ~일 거다” ⭕)
- 객관적 근거, 실제 사건, 타인의 명확한 행동은 포함하지 말 것
- 타인의 생각을 단정하지 말 것 (→ 성급한 결론 금지)
- 성과나 결과의 가치를 깎아내리지 말 것 (→ 긍정 무시/과장·축소 금지)


# 인지왜곡 예시:
- “불안하니까 뭔가 잘못된 게 분명해.”

# 출력 형식:
- CSV 파일
- 각 행은 [text, label] 컬럼으로 구성

# 개수:
총 1200문장 생성

# 추가 조건:
- 문장 상황을 다음 범주에서 고르게 분산할 것
  (일/학업/대인관계/자기평가/미래에 대한 생각)
